# Discoverer_Fin_Controller
Fin controller software for Discoverer Project WP2

Designed for ATmega2560. Runs FreeRTOS 10 with libcsp for comms.

Libraries in use:
FreeRTOS          : https://www.freertos.org/index.html
libcsp            : https://github.com/libcsp/libcsp
arduino-mcp2515   : https://github.com/autowp/arduino-mcp2515
Arduino SPI       : https://github.com/arduino/ArduinoCore-avr/tree/master/libraries/SPI
